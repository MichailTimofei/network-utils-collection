# Contributing to Network Utilities Collection

Thank you for your interest in contributing to Network Utilities Collection!

## How to Contribute

### Reporting Bugs

If you found a bug, please create an issue with the following information:

- A quick summary of the issue
- Steps to reproduce the bug
- What you expected vs what happened
- Browser/OS information

### Suggesting Features

We welcome feature suggestions! Please create an issue with:

- Clear description of the feature
- Use cases for the feature
- Any alternative solutions you've considered

### Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-new-feature`
3. Make your changes
4. Test your changes in a browser
5. Commit with descriptive messages
6. Push to your fork
7. Submit a Pull Request

### Code Style

- Use consistent indentation (2 spaces)
- Keep JavaScript modular and well-commented
- Test across multiple browsers

## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/network-utils-collection.git

# Create a branch
git checkout -b feature/your-feature

# Make changes and test
# Open index.html in your browser

# Commit changes
git add .
git commit -m "Add your feature"

# Push to GitHub
git push origin feature/your-feature
```

## Questions?

Feel free to open an issue for any questions about contributing.
